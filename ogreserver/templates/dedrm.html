Removing DRM

Follow these instructions to get started removing DRM from your Amazon and Adobe purchased ebooks. As usual, it is slightly more complicated for Windows users.

Currently two formats are covered:

  • Amazon Kindle - .AWZ or .MOBI
  • Adobe Digital Editions - .PDF

Amazon Kindle

  1. Download and install Python for Windows:
    • Download and run the Python installer from Activestate.
    • Open a command prompt, by clicking the Start menu and then "Run.."
    • Type this command and hit enter: 
      pypm install pycrypto
  2. Download and install Calibre:
    • Calibre
  3. Download and unzip the DRM Tools for your OS:
    • On OSX, we use a single application called "DeDRM 3.1.app" - OGRE_DRM_Tools_OSX_3.1.zip
    • On Windows, there is a folder containing "DeDRM_Drop_Target.bat" and some other stuff - OGRE_DRM_Tools_Win_2.8.zip
  4. Download and install the Kindle App for your OS:
    • OSX
    • Windows
  5. Setup the Kindle App:
    1. The Amazon application will already show three books. After you have registered, it will also show all the books you've bought.
    2. We need to setup where our books are stored.
      • On Windows this is "My Documents/My Kindle Content".
      • OSX users should goto settings and change the "content folder" where the app downloads to. Remember this folder.
    3. Register the application with your Amazon account.
    4. The little arrow-on-a-cloud icon indicates you haven't downloaded the book yet. Right-click and hit download.
    5. Your downloaded books will be in the folder mentioned above - conveniently named something like 'B005J578EW_EBOK.awz'.
  6. Remove the DRM
    1. Drag and drop your .AWZ file onto your DRM Tools file to clean the DRM:
      • OSX users drop onto DeDRM 3.1.app
      • Windows users drop onto DeDRM_Drop_Target.bat
    2. The output should be a slightly better named file, with the suffix .mobi.
  7. Confirm it worked
    1. Open the Calibre ebook application you installed in step 2. (Your eyes may bleed a little at first, but you'll get used to her).
    2. Drag-drop your newly created .mobi ebook into Calibre.
    3. Double-click the new entry in Calibre, and if it opens for you to read then you're good. You'll get an error message if not..
